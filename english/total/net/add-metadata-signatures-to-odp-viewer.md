steps:
    enable: true
    title_left: "Adding metadata signatures to ODP in .NET"
    content_left: |
        [GroupDocs.Total](https://products.groupdocs.com/total/net/) makes it easy for developers to digitally sign an OpenDocument presentation file (ODP) with metadata signatures using a few lines of C# .NET code.

        *   Instantiate <mark>**Signature**</mark> with input document
        *   Instantiate the <mark>**MetadataSignOptions**</mark> object as per your needs.
        *   Instantiate one or more <mark>**PresentationMetadataSignature**</mark> objects and add them into <mark>**MetadataSignOptions**</mark> to metadata signatures collection (<mark>**Signatures**</mark>) via <mark>**Add**</mark> or <mark>**AddRange**</mark> method.
        *   Call <mark>**Sign**</mark> method and pass <mark>**MetadataSignOptions**</mark> to it.
        *   Set options to view signed document as HTML
        
    title_right: "System Requirements"
    content_right: |
        The below code example requires you to install [GroupDocs.Viewer for .NET](https://products.groupdocs.com/viewer/net/) and [GroupDocs.Signature for .NET](https://products.groupdocs.com/signature/net/) libraries. Also, make sure that you have the following prerequisites ready on your system before executing the code below:

        *   Operating Systems: Microsoft Windows, Linux, MacOS
        *   Development Environments: Microsoft Visual Studio, Xamarin, MonoDevelop
        *   Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        *   Get the latest version of GroupDocs.Total for .NET downloaded from [Nuget](https://www.nuget.org/packages/groupdocs.total)
        
    code: |
        ```cs
        // Add metadata signatures to presentations using GroupDocs.Signature API
        // Instantiate Signature with input ODP file
        using (Signature signature = new Signature("input.odp"))
          {
            // Instantiate the MetadataSignOptions object
            MetadataSignOptions options = new MetadataSignOptions();
          
            // Create few Presentation Metadata signatures
            PresentationMetadataSignature[] signatures = new PresentationMetadataSignature[]
            {
              new PresentationMetadataSignature("Author", "Mr.Scherlock Holmes"),
              new PresentationMetadataSignature("DateCreated", DateTime.Now),
              new PresentationMetadataSignature("DocumentId", 123456),
              new PresentationMetadataSignature("SignatureId", 123.456M)
            };

            // add these signatures to options
            options.Signatures.AddRange(signatures);
            signature.Sign("output.odp", options);
          }

            // Set options to view signed document as HTML
            HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources("output{0}.html");
            try (Viewer viewer = new Viewer("output.odp")) {
              viewer.view(viewOptions);
            }
        ```
